Working with web api controllers

Note that in Asp.net core the apicontroller inherits from controllerbase and the mvc controller inherits from controller. This page talks about api.

  • Return Values from controllers:

    Returning Specific Types

    The simplest action returns a primitive or complex data type (for example, string or a custom object type). Consider the following action, which returns a collection of custom Product objects:

    {{school.name}} {{teacher.name}}

    Returning IActionResult Types

    The IActionResult return type is appropriate when multiple ActionResult return types are possible in an action. The ActionResult types represent various HTTP status codes. Any non-abstract class deriving from ActionResult qualifies as a valid return type. Some common return types in this category are BadRequestResult (400), NotFoundResult (404), and OkObjectResult (200). Alternatively, convenience methods in the ControllerBase class can be used to return ActionResult types from an action
