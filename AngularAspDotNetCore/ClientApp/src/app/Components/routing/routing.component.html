Routing in an .Net Core Angular Application

There are 3 routings used in an angular .net core application. Conventional Routing and Attribute Routing which are server side routings. Additionally there is angular routing which is client side routing. In order for all 3 to co-exist, we need to do an additional configuration on the server side.
